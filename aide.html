Clique Droit

Un clique sur le bouton droit provoque un déplacement du tetrimino d'une case vers la droite

Clique Gauche

Un clique sur le bouton gauche provoque un déplacement du tetrimino d'une case vers la gauche

Clique Bas

Un clique sur le bouton bas provoque un déplacement du tetrimino d'une case vers le bas

Clique Haut

Un clique sur le bouton haut provoque une rotation du tetrimino de 90° dans le sens d'une montre

Clique Sur Espace

Un clique sur le bouton espace provoque un déplacement du tetrimino vers son emplacement approprié

Clique Sur | P |

Un clique sur le bouton P invite à mettre le jeu en pause

Clique Sur | J |

Un clique sur le bouton J invite à utiliser le joker si c'est possible;le joker sert a détruire la dernière ligne
